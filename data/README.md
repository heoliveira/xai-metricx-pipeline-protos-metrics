# Datasets

Coloque os arquivos CSV dos datasets neste diretório antes de executar os experimentos.

| Arquivo                   | Dataset                              |
|---------------------------|--------------------------------------|
| `fhs.csv`                 | Framingham Heart Study               |
| `sepsis.csv`              | Sepsis clinical dataset              |
| `stroke.csv`              | Stroke Prediction Dataset            |
| `uci_thyroid_dxbin.csv`   | UCI Thyroid Disease (binário)        |
| `wdbc.csv`                | Wisconsin Diagnostic Breast Cancer   |

> O dataset **WDBC** é obtido automaticamente via `scikit-learn` se o arquivo CSV não estiver presente.
